Jogo de Adivinhação em Java
Um jogo interativo onde o computador escolhe um número aleatório entre 0 e 100, e o jogador tem 5 chances para acertar.

Como funciona:

O programa gera um número secreto usando a classe Random.

O jogador tem 5 tentativas.

A cada erro, o programa avisa se o número secreto é maior ou menor que o chute atual.

No final, o jogador pode escolher se quer jogar novamente ou sair.

Conceitos aplicados:

Geração de números aleatórios (Random).

Entrada de dados pelo teclado (Scanner).

Controle de repetição com Do-While e For.

Estruturas de decisão (If/Else) para as dicas do jogo.
